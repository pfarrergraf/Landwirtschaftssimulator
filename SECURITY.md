# SECURITY

## Grundsaetze

- Offline zuerst und Datenminimierung.
- Keine E-Mail-Adresse fuer lokale Spielerprofile.
- Keine Werbung und keine Tracking-SDKs im Kinder-MVP.
- Savegames enthalten nur frei gewaehlten Spielernamen, Hofnamen und Spielzustand.
- Keystores und Cloudtokens ausschliesslich als GitHub Secrets.
- Keine geheimen Werte in Issues, Logs, Buildartefakten oder Screenshots.

## Meldung

Sicherheitsprobleme nicht als oeffentliches Issue mit Exploitdetails melden. Vor einem kommerziellen Release wird ein privater Meldeweg eingerichtet.
