# Cloudflare Deployment

## Phase 1

Statische Webseite aus `apps/website/public`. Cloudflare Pages kann direkt mit dem GitHub-Repository verbunden werden. Keine serverseitige App erforderlich.

## Phase 2

Web-Demo als reduzierter Godot-Webexport. Grosse Android-Downloads und Medien koennen in R2 liegen.

## Phase 3

Optionale Workers/D1-Dienste fuer anonyme Klassen-Codes oder Cloudsave. Diese Phase benoetigt Datenschutzkonzept und ist nicht Teil des MVP.
