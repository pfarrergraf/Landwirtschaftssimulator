# Mobile Performance Budget

## Ziel

Stabile 30 FPS auf Android-Mittelklassegeraeten; 60 FPS als Option auf leistungsfaehigen Geraeten.

## Regeln

- Mobile Renderer
- LOD fuer Fahrzeuge und Vegetation
- instanziierte Feldpflanzen statt individueller komplexer Nodes
- Texturatlanten und begrenzte Materialanzahl
- kleine aktive Simulationszone um den Spieler
- entfernte Felder rein datenbasiert simulieren
- Physik nur fuer relevante Fahrzeuge aktiv
- keine hochaufloesenden Markenassets im Basispaket

Exakte Budgets werden nach dem ersten Vertical Slice auf Referenzgeraeten gemessen.
