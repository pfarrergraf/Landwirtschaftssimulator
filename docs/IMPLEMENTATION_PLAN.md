# Umsetzungsplan – erster spielbarer Landwirtschaftsbuild

## Zielzustand

Der erste spielbare Build soll auf Desktop und Android denselben Kernablauf abbilden:

1. Spiel starten und Profil laden.
2. Avatar auf einem kleinen Hof bewegen.
3. Mit einem Traktor interagieren und einsteigen.
4. Ein Testfeld bearbeiten, säen und bis zur Ernte simulieren.
5. Fortschritt speichern, Anwendung neu starten und Zustand wiederherstellen.
