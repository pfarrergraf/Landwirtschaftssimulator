# Felder und Kulturen

Jedes Feld besitzt Groesse, Bodenklasse, Entfernung, Eignung, Feuchtigkeit, Naehrstoffe, Verdichtung, Unkraut, Kultur und Wachstumsstufe.

Im Grundschulmodus werden Werte in Ampeln und drei Stufen uebersetzt. Im Profimodus bleiben numerische Werte sichtbar.

Die zehn Startfelder umfassen zusammen 60 Hektar. Sie unterscheiden sich bewusst, damit nicht jede Kultur auf jedem Feld gleich sinnvoll ist.
