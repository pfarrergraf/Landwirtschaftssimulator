- [ ] Produktname rechtlich geprueft.
- [ ] Marken- und Assetregister vollstaendig.
- [ ] Keine unfreigegebenen Logos oder Fahrzeugmodelle.
- [ ] Datenschutztext und Kinderkonzept geprueft.
- [ ] Keine Tracking-SDKs ohne ausdrueckliche Entscheidung.
- [ ] Alle Tests und Datenvalidierungen gruen.
- [ ] Android-Ziel-API aktuell geprueft.
- [ ] Signierung aus Secrets.
- [ ] Savegame-Migration getestet.
- [ ] Performance auf drei Geraeteklassen getestet.
- [ ] Barrierefreiheitscheck.
- [ ] Storetexte behaupten keine Herstellerpartnerschaft ohne Vertrag.
